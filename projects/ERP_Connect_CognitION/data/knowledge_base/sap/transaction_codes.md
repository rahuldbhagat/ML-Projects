# SAP Transaction Codes Reference

## Frequently Used Transaction Codes

### FB01 - Post Document

**Purpose:** Manual document posting

### FB03 - Display Document

**Purpose:** View posted documents

### FS00 - G/L Account Master

**Purpose:** Create/change G/L accounts

### KS03 - Display Cost Center

**Purpose:** View cost center details

### XK03 - Display Vendor

**Purpose:** View vendor master

### OB52 - Open/Close Periods

**Purpose:** Manage posting periods

### SU53 - Authorization Check

**Purpose:** Check missing authorizations
